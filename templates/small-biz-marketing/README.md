# Small Business Marketing Template

Single-page marketing website template for small businesses. Used by the build script to generate static HTML from Sanity CMS data.

## How it works

The template uses `{{PLACEHOLDER}}` tokens that the build script (`scripts/build.js`) replaces with data from Sanity.

## Placeholder Reference

### Simple strings (direct replacement)

| Placeholder | Sanity Field | Example |
|---|---|---|
| `{{BUSINESS_NAME}}` | `name` | Hajis Famous Deli |
| `{{TAGLINE}}` | `tagline` | Amsterdam Ave · Since the 1990s |
| `{{SEO_TITLE}}` | `seo.title` | Hajis Famous Deli — Morningside Heights, NYC |
| `{{SEO_DESCRIPTION}}` | `seo.description` | Home of the original chopped cheese... |
| `{{DESCRIPTION_DESKTOP}}` | `description` (first paragraph) | For more than thirty years... |
| `{{DESCRIPTION_MOBILE}}` | `descriptionMobile` | The chopped cheese was invented here... |
| `{{PHONE_TEL}}` | `phone` | +16466843003 |
| `{{PHONE_FORMATTED}}` | `phoneFormatted` | (646) 684-3003 |
| `{{ADDRESS}}` | `address` | 1129 Amsterdam Avenue |
| `{{CITY_STATE_ZIP}}` | `cityStateZip` | New York, NY 10025 |
| `{{GOOGLE_MAPS_QUERY}}` | `googleMapsQuery` | 1129+Amsterdam+Ave+New+York+NY+10025 |
| `{{GOOGLE_MAPS_DIRECTIONS_URL}}` | `googleMapsDirectionsUrl` | https://www.google.com/maps/dir/... |
| `{{PAYMENT_LINE}}` | `paymentLine` | Accepts Visa, MasterCard... |
| `{{MENU_NOTE}}` | `menuNote` | This is a selection — the full counter menu has more. |
| `{{YEAR}}` | (auto-generated) | 2026 |

## Customizing the template

Colors and fonts are defined in CSS `:root` variables at the top of the file. To create a different color scheme for a client, you'd modify the CSS in their generated `index.html` after building (or create a variant template).
